== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'RT Facebook Like Box' menu in WordPress.
3. Go to Setting => RT Facebook Like Box => Type your facebook link in url field then Save.
